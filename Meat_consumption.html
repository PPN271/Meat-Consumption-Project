import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np

# Example CO2 emissions data (Gt CO2/year)
# data from 1. Global Greenhouse Gas Emissions Data
data = {
    'Source': ['Electricity and Heat Production ', 'Industry', 'human-caused greenhouse gas emissions', 'Transportation', 'Buildings ', 'Other Energy'],
    'CO2 Emissions': [23, 24, 14.5, 15, 6, 17.5],
}

df = pd.DataFrame(data)
plt.figure(figsize=(10, 6))
plt.pie(df['CO2 Emissions'], labels=df['Source'], autopct='%1.1f%%', startangle=90)
plt.axis('equal')
plt.title('Global Emissions by Economic Sector')
plt.savefig('output/figure_1')
plt.show()

Agriculture, Forestry, and Other Land Use (22% of 2010 global greenhouse gas emissions): Greenhouse gas emissions from this sector come mostly from agriculture (cultivation of crops and livestock) and deforestation. This estimate does not include the CO2 that ecosystems remove from the atmosphere by sequestering carbon in biomass, dead organic matter, and soils, which offset approximately 20% of emissions from this sector.
